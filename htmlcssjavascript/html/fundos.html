Fundos

Mudando a cor do fundo

Ao colocar um fundo verifique se não existe outros elementos da mesma cor, pois caso as cores coincidão, terá uma impressão de que algo não está funcionando ou que sumiu.

Opacidade

Caso deseje fazer uma transparência do fundo, pode-se fazer deste jeito. Cuidado pois ao tentar aumentar depois a sua opacidade pode dar errado por exemplo a palavra a seguir que independente do valor que eu coloque não fica como era antes: Inconstitucionalissimamente ao valor que eu coloque, pórem para diminuir ainda mais é possível sem ter muitos problemas além de ler essa frase fraca.

Imagem de fundo

Diversas configurações devem ser reparadas quando colocamos uma imagem de fundo, para que não ajam problemas de visualização, tais como a repetição da imagem, quando ela acaba, e se o texto é capaz de aparecer sem problemas para leitura.

Para imagens de fundo é possível colocarmos uma ou mais figuras, sendo quase essencial o uso de imagens com fundos vazios, caso contrário pode haver problemas na hora da sobreposição de um fundo pelo outro

Podemos colocar a imagem tanto por link quanto por referência na pasta, se não aparecer uma imagem diferente que a do conteiner anterior que é a imagem de 200x200 é porque o link não serve mais

Tipos de posição

Favor testar todas as suas formas de posições na folha de estilo pois ficaria mais feio do que já está este site se eu tentasse demonstrar cada uma, e a propósito use uma mistura com o background-repeat para poder ter noção de diversos efeitos

Tipos de fixação do fundo

Esse formato é o que vem por padrão e o que ele faz é deixar o fundo junto do elemento

Já desse jeito conforme você vai passando a página o fundo fica fixo lá atrás

Agora a diferença desse é que a sua imagem vai aparecendo conforme o seu elemento é rodado
