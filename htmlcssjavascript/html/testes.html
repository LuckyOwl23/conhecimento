Fundos

Mudando a cor do fundo

Ao colocar um fundo verifique se não existe outros elementos da mesma cor, pois caso isto aconteça terá uma impressão de que algo não está funcionando.

Opacidade

Caso deseje fazer uma transparência do fundo, pode-se fazer deste jeito
Cuidado em seu uso pois os elementos contidos acabam por erdar a opacidade

Imagem de fundo

Diversas configurações devem ser reparadas quando colocamos uma imagem de fundo, para que não ajam problemas de visualização, tais como a repetição da imagem, quando ela acaba, e se o texto é capaz de aparecer sem problemas para leitura.

Tipos de repetição

Repetição de imagem somente lateral

Repetição de imagem somente vertical

Repetição de ambos os sentidos que é o que vem por padrão

Sem repetição

Repetição espaçada tenta fazer a primeira e a última imagem aparecerem sem serem recortadas e depois cria espaços automáticos para distribuir a frequência de aparecimento

Repetição esticada tenta fazer a primeira e a última imagem aparecerem sem serem recortadas e depois adaptam seu espaço para encaixar as outras.
